11) Classification par mots-cles (classifier.py)
- Fonctions pures implementees:
  detect_region(text), detect_theme(text), detect_niveau(text, valeur_pm25=None)
- Normalisation des accents et de la casse pour fiabiliser les matchs.
- Seuil PM2.5 respecte:
  >150 critique, >55 eleve, >12 modere, sinon info.

12) Scraping multi-sources (scraper.py)
- Sources prises en charge:
  - UNEP News (avec priorite RSS UNEP, fallback page)
  - Reuters Environment (page officielle + fallback RSS Google News filtre Reuters)
  - Notre Planete Info (pages actualites detaillees)
  - OpenAQ v2 (URL demandee) + fallback v3 si necessaire
- Pour chaque source:
  timeout=8s, try/except local, logs d'erreur, continuation des autres sources.
- Transformation article:
  - id = hashlib.md5(titre.encode("utf-8")).hexdigest()[:12]
  - classification region/theme/niveau via classifier.py
  - insertion dedoublonnee via database.py
- Description normalisee entre 100 et 200 mots.

13) Scheduler APScheduler (scheduler.py)
- BackgroundScheduler configure avec:
  - scrape_all() toutes les 24h
  - update_stats() toutes les heures
- Au demarrage:
  - si base vide -> scrape_all() immediat
  - sinon -> update_stats()
- Arret propre avec atexit (shutdown du scheduler).
